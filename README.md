# Awesome Developer Podcasts

A curated directory of the best podcasts for programmers, web developers, and software engineers.

| Podcast Name | Description | Website Link |
| --- | --- | --- |
| **How About Tomorrow?** | Adam and Dax discuss web technologies, the internet, and the future of (basically) everything. | [tomorrow.fm](https://tomorrow.fm/) |
| **Over Engineered** | A podcast where we explore unimportant programming questions (mostly PHP/Laravel/JavaScript) in extreme detail. | [overengineered.fm](https://overengineered.fm/) |
| **Scaling DevTools** | Interviews with the people behind established DevTools like Algolia, SendGrid & AWS. As well as up-and-coming DevTool startups like Neptune.ai and AssemblyAI to find out how side projects become unicorns. | [podcast.scalingdevtools.com](https://podcast.scalingdevtools.com/) |
| **No Compromises** | Two seasoned salty programming veterans talk about best practices based on years of working with Laravel SaaS teams. | [show.nocompromises.io](https://show.nocompromises.io/) |
| **The Laravel Podcast** | The Laravel Podcast brings you Laravel and PHP development news and discussion. | [laravelpodcast.com](https://laravelpodcast.com/) |
| **The Scrimba Podcast** | Interviews with successful developers about how they learned to code and got their first job. | [scrimba.com/podcast](https://scrimba.com/podcast/) |
| **Campfire Coders** | Two developers chatting about the tools, technologies, and practices they use every day. | [campfirecoders.com](https://campfirecoders.com/) |
| **Indie Rails** | A podcast about Ruby on Rails, indie hacking, and building SaaS products. | [indierails.com](https://indierails.com/) |
| **Mostly Technical** | A weekly podcast where Aaron and Ian discuss mostly technical topics. | [mostlytechnical.com](https://mostlytechnical.com/) |
| **No Plans to Merge** | A podcast where two developers talk about code, life, and everything in between. | [noplanstomerge.com](https://noplanstomerge.com/) |
| **Syntax.fm** | A Tasty Treats Podcast for Web Developers. | [syntax.fm](https://syntax.fm/) |
| **Hackers Incorporated** | A podcast about building and bootstrapping software companies. | [hackersincorporated.com](https://hackersincorporated.com/) |
| **North Meets South Web Podcast** | A podcast about web development, PHP, JavaScript, and everything in between. | [northmeetssouth.audio](https://northmeetssouth.audio/) |
| **FSJam Podcast** | A podcast about Fullstack Jamstack and the future of web development. | [fsjam.org](https://fsjam.org/) |
| **The Technium** | A podcast about technology, software development, and the future. | [thetechnium.fm](https://thetechnium.fm/) |
| **Developers! - mer än bara kod** | En svensk podcast om mjukvaruutveckling och allt runtomkring. | [developerspodcast.com](https://developerspodcast.com/) |
| **DevOps and Docker Talk** | Interviews and discussions about Docker, DevOps, containers, and cloud native technologies. | [podcast.bretfisher.com](https://podcast.bretfisher.com/) |
| **APIs You Won't Hate** | A podcast about API design, development, and everything in between. | [apisyouwonthate.com](https://apisyouwonthate.com/) |
| **Evergreen podcasts** | A collection of timeless podcasts for developers. | [transistor.fm/evergreen](https://transistor.fm/evergreen/) |
